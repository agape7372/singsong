# 티켓(표) 디자인 — 추구미 참고

> **이 폴더를 반드시 참고할 것.** 앞으로 세션 티켓/표 디자인 작업(코드·프롬프트·리디자인) 시
> 아래 추구미와 `채택/`·`미채택/` 예시로 방향을 고정한다. 모든 이미지는 reference-only(정본 §10) —
> 실제 UI는 코드와 제어된 inline SVG로 재구성한다.

## 추구미 (지향하는 미감)

- **모던 + 소장욕구**: 밋밋·평범 금지. 갖고 싶고 모으고 싶은 티켓.
- **깔끔하고 대담**: 볼드한 타이포·기하, 넉넉한 여백. **멤피스st 경쾌한 기하 에너지 환영**.
- **세로 펀치티켓 구조 유지**: 양옆 펀치홀 컬럼 · 절취선 · SN/바코드 스텁 · 큰 곡수 히어로.
- **팔레트**: 크림 바탕 · 잉크(오버진 `#15131A`) · **핑크 액션 `#FF3D6E`** · 오커 머니(`#B76E00`).
- **콘텐츠**: 곡수 · 시간 · 총액 · 1인당 · serial(요약 아티팩트).

## ⚠ 프롬프트 규칙 — "rose/장미" 금지

- 색상 이름으로 **"rose" 쓰지 말 것** → 모델이 **장미꽃·장미 왁스씰**을 그려 촌스러워짐(gacha 우하단 사례).
- 색은 항상 **"pink `#FF3D6E`"**(또는 magenta-pink)로 표기.
- **장미·꽃·왁스씰·엠블럼 seal 금지** (꽃 이미지 자체 배제).

## 금지 (미채택 사유의 공통분모)

- 옛날감성/레트로 촌스러움 (빈티지 승차권·신문 지면 등).
- 눈아픈 과밀 · 강한 검은 테두리 박스 · 잡다한 공예풍 (자수·스테인드글라스·바이닐 등).
- 밋밋·평범 (초기 미니멀·일부 모던안).
- **장미·꽃·왁스씰** (위 규칙).
- 그라디언트 · 글래스 · 네온 · 3D · 사실적 텍스처.
- 마이크 · 음표 · 헤드폰 클리셰.
