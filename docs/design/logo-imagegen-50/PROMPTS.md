# SingSong app-logo image-generation set — 50 directions

## Shared brief

Apply this brief to every numbered modifier below.

```text
Use case: logo-brand
Asset type: mobile app icon symbol exploration for SingSong, a Korean karaoke session planner that turns a shared song queue, estimated time, and cost into one confirmed ticket.
Primary request: Create ONE original abstract logo mark using the numbered concept modifier. Express planning, ordering, gathering, calculation, or issuance without a music-playback cliché.
Style/medium: exceptionally clean vector-like logo mark; flat solid geometry; precise edges; strong silhouette; professional identity-system quality; scalable and intelligible in one color.
Composition/framing: exactly one centered symbol on a square 1:1 canvas; generous margin; the complete mark stays within the central 66% maskable safe zone; readable at 24 px.
Color palette: warm cream background #FAF7F0; use near-black #15131A and action rose #FF3D6E for the mark. Use restrained ochre #B76E00 only when the modifier explicitly requests a settlement or confirmation point. Maximum three mark colors.
Constraints: no wordmark, no letters, no Hangul, no labels, no numbers, no border, no app/device mockup, no gradients, no shadows, no texture, no 3D, no watermark. No music note, microphone, headphones, play triangle, waveform, equalizer, speaker, heart, star, sparkle, confetti, neon, or generic decorative ribbon. Avoid tiny details. Deliver exactly one isolated logo concept, never a grid or presentation sheet.
```

## Review gate

Reject or regenerate any result that includes forbidden imagery, extra text, more than one logo, a presentation board, obvious shadows/gradients/3D, or a mark that collapses at thumbnail size. Generated PNGs are exploration artifacts; any finalist must later be redrawn as a controlled SVG master.
