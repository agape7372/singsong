# SingSong 앱 디자인 시안 10종

모바일 플랜 홈을 동일한 샘플 상태로 고정하고, 레이아웃·타이포·색·상호작용 리듬을 10가지 방향으로 탐색한 비교 보드입니다.

## 채택 결정

- **우선 채택안**: `CUTLINE — 에디토리얼 발권지`
- **상태**: 채택, 첫 번째 구현 슬라이스 이후 사용성 검증
- **결정일**: 2026-07-21
- **결정 기록**: [`ADOPTION.md`](./ADOPTION.md)

나머지 9개 시안은 구현 기준이 아닌 향후 실험·참고안으로 유지합니다.

## 실행

`index.html`을 브라우저로 열면 별도 설치 없이 동작합니다.

- 상단 필터: 라이트 / 다크 / 실험형 분류
- 각 시안 우측 상단 화살표: 풀 스케일 미리보기
- CTA와 탭: 비교 보드용 상태 안내
- 반응형: 데스크톱 5열부터 모바일 1열까지 대응

## 고정한 제품 상태

- 8곡
- 시간 요금 `30분 / 10,000원`
- 4명
- 약 `20–40분`
- 총 `10,000–20,000원`
- 1인당 약 `2,500–5,000원`

곡명은 실제 카탈로그가 아닌 합성 테스트 데이터입니다.

## 방향별 참고

1. `CUTLINE`: **채택안** — 정본과 구현 적합도가 가장 높음
2. `THREE BEATS`: 처음 쓰는 사람에게 가장 명확한 참고안
3. `POSTER STACK`: 공유 화면의 인상이 강한 참고안
4. `NIGHT ROUTE`: 다크 모드 탐색용 참고안

나머지 방향은 특정 사용자 톤 또는 시그니처 인터랙션을 검증하는 탐색안입니다.
